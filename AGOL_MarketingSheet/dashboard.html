Interactive AGOL Analytics

This project involved the development of an engaging, on-demand visual analytics platform using the ArcGIS Online (AGOL) ecosystem. The core deliverable was a highly customized ArcGIS Dashboard, embedded within a narrative-driven ArcGIS StoryMap, designed to facilitate user-led spatial data exploration.


The chosen NRCan dataset covered wildfires across Canada from 1972-2025, presenting a comprehensive view of fire patterns and spatial trends. The challenge was how to effectively filter and display the data in a meaningful way with the tools available.


The workflow required sourcing cloud-based feature layers, configuring dynamic Web Maps with specialized symbology, and employing interactive dashboard elements in a meaningful way. This integrated approach prioritized user experience, ensuring that complex spatial relationships were communicated clearly to non-technical audiences.
